# ExampleTech - Machine Learning Engineer

ExampleTech is hiring a Machine Learning Engineer to help build production machine learning systems for data-intensive products.

Responsibilities include:

- Develop Python services for model training, evaluation, and inference.
- Build reliable data pipelines and validation checks for structured and semi-structured data.
- Evaluate models using appropriate offline and online metrics.
- Collaborate with software engineers to integrate ML components into backend systems.
- Improve reproducibility across experiments and production deployments.
- Work with cloud infrastructure, Docker, CI/CD, and MLOps tooling.

Useful experience:

- Python, machine learning, and strong software engineering fundamentals.
- PyTorch, scikit-learn, Pandas, NumPy, or similar tools.
- Model evaluation, data quality checks, and experiment tracking.
- APIs, testing, version control, and cloud deployment.
