# Raw Inputs

Use this folder for source material that helps bootstrap or update `data/master.yaml`.

Good candidates include:

- copied text from old CVs
- LinkedIn profile exports or copied profile text
- project notes
- GitHub README excerpts
- certification notes
- rough career notes
- text extracted from old PDF CVs

Do not treat this folder as the source of truth. Raw inputs are only staging material. Once useful information has been extracted, normalized, and reviewed into `data/master.yaml`, the master YAML becomes the source of truth for generated CVs.

PDF parsing is not implemented in this MVP. If you have old PDF CVs, convert them to text manually or with a separate tool, then place the extracted text here.
